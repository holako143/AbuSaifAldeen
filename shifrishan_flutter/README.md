# Shifrishan Flutter (شفرشان الأصيل)

هذه هي النسخة الأصيلة (Native) من مشروع شفرشان، تم بناؤها باستخدام Flutter لضمان أقصى درجات الأمان والخصوصية على أجهزة الجوال.

## المميزات التقنية:
*   **تشفير AES-256-GCM:** متوافق تماماً مع نسخة الويب.
*   **إخفاء البيانات في الرموز:** استخدام تقنية Variation Selectors المتقدمة.
*   **إخفاء البيانات في الصور:** معالجة أصيلة للبكسلات (LSB) تدعم الصور عالية الدقة.
*   **التخزين الآمن:** استخدام Hive المشفر و Secure Storage للمفاتيح الحساسة.

## الميزات الجديدة في نسخة فلاتر:
1.  **المصادقة الحيوية:** فتح الخزنة ببصمة الإصبع أو الوجه.
2.  **حماية الشاشة:** منع أخذ لقطات شاشة (Screenshots) داخل التطبيق.
3.  **كلمة سر الطوارئ:** نظام مسح البيانات الفوري.
4.  **الخزنة التمويهية:** دعم الإنكار المقبول عبر كلمة سر ثانية.

## هيكلية المشروع:
*   `lib/core/`: يحتوي على خدمات التشفير، إخفاء البيانات، والأمان.
*   `lib/views/`: واجهات المستخدم (الرئيسية، الخزنة، الصور).
*   `lib/providers/`: إدارة حالة التطبيق (Riverpod).
*   `lib/widgets/`: المكونات القابلة لإعادة الاستخدام.

## طريقة التشغيل:
1. تأكد من تثبيت Flutter SDK.
2. قم بتشغيل `flutter pub get` لتحميل المكتبات.
3. قم بتشغيل التطبيق باستخدام `flutter run`.
